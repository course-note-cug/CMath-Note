1 回顾变量的微分学

在学习向量和矩阵的求导之前, 最好回顾关于一元函数、多元函数的微分学.
一元函数的微分学
微分算子 D 是一个线性算子. 这就意味着对于映射 f,g 、常数 k,l , 有 D( kf+lg ) =kD( f ) +lD( g ) . 此外, 我们还有链式法则, 即 f( g( x ) ) =f ' g( x ) g ' x .
多元函数的微分学
最重要的是考虑多元函数的复合函数.

2 向量求导: 两种格式

一句话概括: 向量求导无非就是将多个变量排成了一个比较好的格式. 并且在这个比较好的格式上面求微分. 再继续之前, 我们先来回顾几个重要的技巧:
  1. 作为点积的求和. 对于形如 ∑ i u i v i 的式子, 可以使用两个列向量 u , v 并且写作点积的形式: 即 ∑ u i v i = u ⋅ v = u'v .
  2. 作为矩阵乘法的点积和. 对于一个矩阵 X , 其中如果第 i 行第 j 列的元素是 u i ⋅ v j , 可以定义两个矩阵 U =[ | u 1 | | u 2 | ⋯ | u n-1 | | u n | ] 、 V =[ | v 1 | | v 2 | ⋯ | v n-1 | | v n | ] , 使得 X=U'V .

2.1 基本定义

定义1. (向量对标量求导) 向量 y =[ y 1 y 2 y 3 ⋯ y m ]' 对于标量 x 的求导记作 d y d x := [ d y 1 d x d y 2 d x d y 3 d x ⋯ d y m d x ] ' .
注. (1) 这通常称为向量 y 的切向量. 实际上这里映射 y : R 1 → R m .
(2) 这里我们采用竖着排版的形式. 这样的情形只是一个习惯性操作. 倘若定义 d y d x :=[ d y 1 d x d y 2 d x d y 3 d x ⋯ d y m d x ], 那么后续的内容除了符号上有更改以外, 没有太多区别.
定义2. (标量对向量求导) 标量 y 对于向量 x =[ x 1 x 2 x 3 ⋯ x m ]' 的分子形式记作
∂ y ∂ x :=[ d y d x 1 d y d x 2 d y d x 3 ⋯ d y d x m ] . 其分母形式可以类似记作 ∂ y ∂ x :=[ d y d x 1 d y d x 2 d y d x 3 ⋯ d y d x m ]'.
这里分子形式的意思是: 生成的向量的行数和分子的行数相同.
注. (1) 分子形式和分母形式实际上是求导之后对于变量如何排布进行的约定. 如果不加说明, 下列总是使用分子形式.
(2) 实际上这是映射 y : R n → R . 并且与梯度之间的关系是 ∇ y=( ∂ y ∂ x ) ' .
定义3. (向量对向量求导) 向量 y= [ y 1 , y 2 , ⋯ , y m ] ' 对向量 x= [ x 1 , x 2 , ⋯ , x n ] ' 求导的分子形式写作 ∂ y ∂ x :=[ ∂ y 1 ∂ x 1 ∂ y 1 ∂ x 2 ⋯ ∂ y 1 ∂ x n ∂ y 2 ∂ x 1 ∂ y 2 ∂ x 2 ⋯ ∂ y 2 ∂ x n ⋮ ⋮ ⋱ ⋮ ∂ y m ∂ x 1 ∂ y m ∂ x 2 ⋯ ∂ y m ∂ x n ] m 行 n 列
其分母形式就是分子形式的转置.
这里分子形式的意思是: 生成的矩阵的行数和分子的行数相同.
注. (1)实际上映射 y : R m → R n , 这就是Jacobian矩阵.
定义4. (矩阵对标量求导) 矩阵 Y ∈ R m × n 对于标量 x 的求导的分子形式记作 ∂ Y ∂ x = ( ∂ y i,j ∂ x ) i,j .
分母形式就是分子形式的转置.
这里分子形式的意思是: 生成的矩阵的行数和分子的行数相同.
注. 这时候 Y : R → R m × n .

定义5. (标量对矩阵求导) 标量 y 对于矩阵 X ∈ R p × q 求导的分子形式记作 ∂ y ∂ x = ( ∂ y ∂ x ) i,j .
分母形式就是分子形式的转置.
这里分子形式的意思是: 生成的矩阵的行数和分母的列数相同. (因为分子就是一个标量).
注. 实际上这时候映射 y: R p × q → R . 常见的函数为矩阵的迹、矩阵的行列式.

3 常见求导结果

3.1 向量对向量

  1. 常数相关
    1. 常向量的导数是 0: 若 a 中不含变量 x , 那么 ∂ a ∂ x = 0 .
    2. 常数进出求导: 若 a 中不含变量 x , u=u ( x ) 那么 ∂ a u ∂ x =a ∂ u ∂ x .
    3. 常矩阵进出求导:
      1. 若 A ∈ R n × m 中不含变量 x ∈ R m × 1 , 那么 ∂ Ax ∂ x = A .
      2. 若 A ∈ R n × m 中不含变量 x ∈ R m × 1 , 那么 ∂ x'A ∂ x = A' .
      3. 若 A ∈ R n × m 中不含变量 x ∈ R m × 1 , 而向量值函数 u=u( x ) ∈ R m × 1 , 那么 ∂ Au ∂ x = A .
  2. 乘法
    1. 变元在向量: a 中不含 x , 但是 u = u ( x ) : ∂ a u ∂ x =a ∂ u ∂ x .
    2. 变元在常数: v=v( x ) , a 中不含 x : ∂ v a ∂ x = a ∂ v ∂ x .
    3. 两个都有: v=v( x ) , u=u( x ) : ∂ v u ∂ x =v ∂ u ∂ x + u ∂ v ∂ x .
  3. 加法: u=u( x ) ,v=v( x ) , 那么 ∂ ( u+v) ∂ x = ∂ u ∂ x + ∂ v ∂ x .
  4. 链式法则
    1. u=u( x ) ,g=g( u ) , ∂ g ∂ x = ∂ g ∂ u ∂ u ∂ x .
    2. u=u( x ) ,f=f( g ) ,g=g( u ) , ∂ f ∂ x = ∂ f ∂ g ∂ g ∂ u ∂ u ∂ x .

3.2 标量对向量

  1. 常数相关: a 中不含 x , u=u( x ) . ∂ au ∂ x =a ∂ u ∂ x .
  2. 加法: u=u( x ) ,v=v( x ) . ∂ ( u+v ) ∂ x = ∂ u ∂ x + ∂ v ∂ x .
  3. 乘法: 如果 u=u( x ) ,v=v( x ) , ∂ uv ∂ x =u ∂ v ∂ x +v ∂ u ∂ x .
  4. 链式法则: f=f( g ) ,g=g( u ) ,u=u( x ) , ∂ f ∂ x = ∂ f ∂ g ∂ g ∂ u ∂ u ∂ x .
  5. 点积: u=u( x ) ,v=v( x ) , ∂ ( u ⋅ v ) ∂ x = u ' ∂ v ∂ x + v ' ∂ u ∂ x .
  6. 二次型: u=u( x ) ,v=v( x ) ,A 中不含 x , ∂ ( u'Av ) ∂ x = u ' A ∂ v ∂ x + v ' A ' ∂ u ∂ x .

3.3 向量对标量

  1. 常数相关:
    1. 常数: a 与 x 无关, u=u ( x ) . ∂ a u ∂ x =a ∂ u ∂ x .
    2. 常矩阵: A 与 x 无关, u=u ( x ) . ∂ Au ∂ x = A ∂ u ∂ x .
  2. 转置: 如果 u=u ( x ) , 那么 ∂ u ' ∂ x =( ∂ u ∂ x ) ' .
  3. 加法: u=u ( x ) , v=v ( x ) . 那么 ∂ ( u+v) ∂ x = ∂ u ∂ x + ∂ v ∂ x .
  4. 乘法相关
    1. 向量的外积: u=u ( x ) , v=v ( x ) . ∂ ( u' × v ) ∂ x =( ∂ u ∂ x ) ' × v + u ' × ∂ v ∂ x .
    2. 矩阵的外积: ∂ ( U × v ) ∂ x = ∂ U ∂ x × v + U × ∂ x ∂ x .
  5. 链式法则: u=u ( x ) , f=f( g ) ,g=g( u ) . 那么 ∂ f ∂ x = ∂ f ∂ g ∂ g ∂ u ∂ u ∂ x .

3.4 向量对矩阵

  1. 常数相关: a 不含 X 相关的变量, u=u( X ) . 那么 ∂ au ∂ X =a ∂ u ∂ x .
  2. 加法: 如果 u=u( X ) ,v=v( X ) . 那么 ∂ ( u+v ) ∂ X = ∂ u ∂ X + ∂ v ∂ X .
  3. 乘法: u=u( X ) ,v=v( X ) . ∂ uv ∂ X =u ∂ v ∂ X +v ∂ u ∂ X .
  4. 链式法则:
    1. u=u( X ) , ∂ f( g( u ) ) ∂ X = ∂ f( g ) ∂ g ∂ g( u ) ∂ u ∂ u ∂ X .
    2. U = U ( X ) , ∂ g( U ) ∂ X ij = tr ( ∂ g( U ) ∂ U ∂ U ∂ X ij )

3.5 矩阵对标量

  1. 常数相关:
    1. 标量常数: U = U ( x ) ,a 不是 x 的函数: ∂ a U ∂ x =a ∂ U ∂ x .
    2. 矩阵: A,B 不是 x 的函数, U=U ( x ) , ∂ AUB ∂ x = A ∂ U ∂ x B .
  2. 加法: U = U ( x ) , V = V ( x ) , ∂ ( U + V ) ∂ x = ∂ U ∂ x + ∂ V ∂ x .
  3. 乘法相关: U=U ( x ) , V=V ( x )
    1. 矩阵乘法: ∂ ( U V ) ∂ x = U ∂ V ∂ x + ∂ U ∂ x V .
    2. 张量乘法: ∂ ( U ⊗ V ) ∂ x = U ⊗ ∂ V ∂ x + ∂ U ∂ x ⊗ V .
    3. 逐元素相乘: ∂ ( U ○ V ) ∂ x = U ○ ∂ V ∂ x + ∂ U ∂ x ○ V
    4. 矩阵的逆:
      1. 一元情况: ∂ U -1 ∂ x =- U -1 ∂ U ∂ x U -1
      2. 多元情况: U = U ( x,y ) , ∂ 2 U -1 ∂ x ∂ y = U -1 ( ∂ U ∂ x U -1 ∂ U ∂ y - ∂ 2 U ∂ x ∂ y + ∂ U ∂ y U -1 ∂ U ∂ x ) U -1
  4. 函数的复合: A 不是 x 的函数, g( X ) 是标量形式的多项式(可以是无穷级数的形式); g'( x ) 是对应的标量求导的结果; g'( x ) 是对应矩阵的变化, 有 ∂ g ( x A ) ∂ x = A g ′ ( x A ) = g ′ ( x A ) A .
